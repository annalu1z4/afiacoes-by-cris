Configuração inicial
Para salvar os dados, o app precisa de uma Planilha Google. Siga os passos abaixo — só precisa fazer isso uma vez!
1Acesse sheets.google.com e crie uma planilha nova. Pode chamar de "Afiação Alicates".
2Copie o ID da planilha: é a parte longa da URL entre /d/ e /edit.
docs.google.com/spreadsheets/d/ID_AQUI/edit
3Na planilha, clique em Extensões → Apps Script. Cole o script gerado abaixo e publique como app web.
